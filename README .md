#  Realtime Multi-Client Chat Application (Python)

A WhatsApp-like **realtime chat application** built using **Python
socket programming** and **Tkinter GUI**.\
This project demonstrates core concepts of **computer networking**,
**client--server architecture**, and **multithreading**, making it ideal
for beginners and academic mini-projects.

------------------------------------------------------------------------

##  Project Description

This application enables multiple users (clients) to connect to a
central server and exchange text messages in real time.\
Each client runs a graphical interface, while the server handles all
connections and message broadcasting.

The project focuses on **real-time communication** without using a
database, keeping the implementation simple and easy to understand.

------------------------------------------------------------------------

##  Technologies Used

##  Features

-   Multi-client chat support\
-   Realtime message broadcasting\

## ▶ How to Run the Project

### Step 1: Start the Server

``` bash
python server.py
```

### Step 2: Start the Client

``` bash
python client.py
```

Run multiple clients by opening multiple terminals.
